class SimitinfoSpider(scrapy.spiders.Spider)
    SimitinfoSpider(*args, **kwargs)
 
This class allows to scrape the simit page.
simit: information system that consolidates the affiliations that report
the entities and administrators of the Social Protection System, according
with what is established in the current regulation.
 
Returns the data of a person according to the initial parameters that are sent.
 
To consult the data of a person using POST, the service must be started
from scrapyrt like this: scrapyry -p 9081.
 
Then, the data that is sent by POST must have the following structure:
    {
        "jsonUserData": '{"rowId":1,"identificationType":"5|CC","identityNumber":"555555","expeditionDate":"20\/09\/2012"}',
        "spider_name": "simit",
        "start_requests": "True"
    }
